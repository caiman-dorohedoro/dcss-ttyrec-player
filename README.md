## Features

- [x] Play .ttyrec and .ttyrec.bz2 files
- [x] Playlist
- [x] Merge multiple ttyrecs (works even when ttyrec and ttyrec.bz2 files are mixed)
- [x] Search (!!, supports regular expressions)
- [x] i18n - Korean, English
- [x] Cache bz2 unzipped results
